Your work in this project will go exclusively in classes GeneticCars.java and Car.java.  You do not need to modify any other file, or make any additional classes.


Feel free to use Eclipse or IntelliJ, but if you want to compile everything on the command line, type
	javac *.java

To run this program, use main class GeneticCars
	java GeneticCars

To test the physics simulator, use the main class Interactive
	java Interactive
